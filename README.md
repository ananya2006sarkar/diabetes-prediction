# diabetes-prediction
A machine learning project that predicts whether a patient is likely to develop diabetes based on medical diagnostic data.

##  Dataset
Pima Indians Diabetes Dataset (available on Kaggle)

## Tech Stack
- Python, pandas, NumPy
- scikit-learn (Logistic Regression, Random Forest)
- Matplotlib, Seaborn

## How It Works
1. Load and explore the dataset
2. Handle missing/zero values and scale features
3. Train classification models
4. Evaluate using accuracy, confusion matrix, classification report

## Results
- Random Forest Accuracy: ~78%
- Key features: Glucose, BMI, Age
